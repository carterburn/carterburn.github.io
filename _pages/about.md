This site is a place for me to brain dump some thoughts. My goal is to just
write technical content as unfiltered / unedited as possible. One of my biggest
hurdles to getting written content public is continual refinement of a post that
is already 90% there. That's what this blog / site is going to be now...a
collection of 90% posts. I'm going to try to write a quick blurb on what I'm
working on currently every 1-2 weeks and not care too much if it's perfectly
edited. I'll give myself roughly 45 minutes each post to just write and get it
out there. Hic sunt dracones!

## About Me
I've spent the last ~7 years in the security world. I've participated in what
would be consider red teaming, developing (mostly offensive) security tools, and
instructing newcomers to the security world. I'm currently seeking a career
refocus to the engineering side of the house, using the practical security
lessons I've learned. I'm experienced in Rust (where most of the tools were
written) and very interested in low-level systems programming and distributed
systems problems.
